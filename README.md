# Lista de Tarefas em React

Um projeto simples de **lista de tarefas (To-Do List)** feito com **React** e **Tailwind CSS**, que permite adicionar, concluir e deletar tarefas de forma dinâmica.

---

## Funcionalidades

1. **Adicionar Tarefas**  
   - Usuário pode digitar uma nova tarefa no input e clicar em "Adicionar".
   - Cada tarefa recebe um ID único incremental.

2. **Marcar Tarefas como Concluídas**  
   - Ao clicar no botão de concluir, a tarefa é riscada e o texto fica cinza.  
   - É possível alternar entre concluída e não concluída.

3. **Deletar Tarefas**  
   - Ao clicar no botão de deletar, a tarefa é removida da lista.

4. **Lista Dinâmica**  
   - A lista acompanha as tarefas adicionadas.  
   - Caso não existam tarefas, exibe a mensagem: `"Adicione novas tarefas!"`.

5. **UI Responsiva e Estilizada**  
   - Usando Tailwind CSS, o layout é responsivo e visualmente agradável.  
   - As tarefas têm um estilo clean com bordas arredondadas e sombra.

---

## Tecnologias Utilizadas

- **React** – Biblioteca para construção de interfaces.  
- **Tailwind CSS** – Framework CSS para estilização rápida e responsiva.  
- **JavaScript ES6+** – Lógica de adição, conclusão e exclusão de tarefas.
